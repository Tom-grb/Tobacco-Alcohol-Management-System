# 烟酒管理系统需求文档

# 一、APP基础功能 - 用户账户管理

## 1. 账号注册

支持新用户注册功能，注册需填写必填信息（用户名、密码、邮箱），密码需满足复杂度要求（如8-20位，包含字母、数字及特殊字符），用户名需校验唯一性，避免重复注册。

## 2. 账号密码登录

系统支持账号+密码的登录方式，登录鉴权需具备基础的密码加密（如MD5/BCrypt），保障账户安全。


# 二、香烟管理模块

## 1. 核心数据记录

需记录单条香烟的全量信息，字段及规则如下：

|字段名称|字段说明|特殊要求|
|---|---|---|
|香烟名称|商品全称（如：软中华（硬盒））|支持模糊查询，非空|
|SN码|商品唯一识别码|唯一可空，作为扫码查询主键|
|编码|公司统一商品编码|唯一非空，作为批量更新匹配依据|
|收货价|实际收货成本价|数值型，自动记录每次更新时间戳|
|批发价|对外批发定价|数值型，自动记录每次更新时间戳|
|零售价|终端销售定价|数值型，自动记录每次更新时间戳|
|香烟图片|商品实物图|香烟图片<br>商品实物图<br>支持APP内拍照上传或从相册选取，上传前自动压缩，限制单张大小≤2MB，格式限定为JPG/PNG，支持预览、删除，上传后生成适配移动端的缩略图。|
|创建时间|数据录入系统时间|自动生成，不可手动修改|
## 2. 批量数据更新

支持导入公司标准香烟文档（格式限定为Excel/CSV），系统按“编码”字段匹配现有数据；仅当文档中的批发价与系统存储的批发价不一致时，才执行更新操作，更新后自动覆盖原有批发价并记录本次更新时间戳。导入文件提取文件的商品编码、商品、批发价、参加名称进行导入。

## 3. 数据导出

支持导出功能自定义配置，用户可按需勾选字段导出（可选字段：SN码、商品编码、商品名称、批发价、收货价、零售价、金额、厂家名称），导出格式为Excel/CSV；同时保留全量导出选项，全量导出内容包含所有核心字段及各价格的最新更新时间，满足数据备份、统计报表等多样化需求。

## 4. 查询功能

- 名称查询：支持香烟名称模糊查询，查询结果可按收货价、批发价、零售价、更新时间排序展示，分页加载。

- 扫码查询：支持调用APP摄像头扫描SN码，快速定位对应香烟，直接跳转至该香烟详情页面，提升移动端查询效率。

## 5. 详情展示与基础操作

提供单个香烟独立信息展示页面，完整呈现所有核心字段、商品图片及各价格的更新历史（含价格数值与对应时间戳）；页面内置新增、编辑、删除按钮，支持全流程增删改查操作，删除操作需二次确认，避免误删。

# 三、酒类管理模块



## 1. 核心据记录（按批次管理）

需记录每批次酒的全量进货信息，字段及规则如下：

|字段名称|字段说明|特殊要求|
|---|---|---|
|酒名|酒类商品全称（如：53度飞天茅台（500ml））|支持模糊查询，非空|
|编码|自动生成|唯一非空，作为数据匹配、批量操作的核心依据|
|SN码|批次唯一识别码|唯一可空，作为扫码查询主键|
|箱数量|本次进货的总箱数|正整数型，非空|
|每箱瓶数|单箱包含的酒瓶数量|正整数型，非空|
|单瓶价格|单瓶进货成本价|数值型，非空|
|总金额|本次批次进货总费用|自动计算（箱数量×每箱瓶数×单瓶价格），不可手动修改|
|供货商|本次进货的供应商名称|支持下拉选择或手动输入，非空|
|进货时间|本次批次进货日期时间|可手动选择，默认当前时间，非空|
|酒图片|批次商品实物图|支持APP内拍照上传或从相册选取，上传前自动压缩，限制单张大小≤2MB，格式限定为JPG/PNG，支持预览、删除|


## 3. 查询功能

- 名称查询：支持酒类名称模糊查询，查询结果可按进货时间、单瓶价格、总金额排序展示，分页加载。

- 扫码查询：支持调用APP摄像头扫描SN码，快速定位对应批次酒，直接跳转至该批次酒详情页面，适配移动端操作习惯。

## 4. 数据导出

支持导出功能默认导出字段为：酒名、单瓶价格、进货时间；用户可选择时间全量导出按照时间顺序导出（编码、SN码、箱数量、每箱瓶数、总金额、供货商），导出格式为Excel/CSV。

## 2. 价格可视化功能

酒类详情页需展示该酒图片，同时支持呈现该酒历史前10次进货的单瓶价格可视化折线图。图表需标注每次进货的核心信息（单瓶价格、进货时间、供货商名称），APP端支持点击图表数据点查看详情；页面同步展示该酒最新一次进货的单瓶价格，及历史入库记录列表，每条记录需包含箱数量、每箱瓶数、总金额、供货商及进货时间，清晰呈现批次流转轨迹。且支持增加和删除入库记录。

# 四、通用规则说明

1.  所有模块的数据新增、编辑需做基础字段校验（如数值合法性、非空项校验），校验不通过时提示明确错误信息。

3.  分页功能统一设置默认每页展示条数（如20条），支持手动调整页码及每页条数。
